## About the Exam

The [AWS Certified Cloud Practitioner](https://aws.amazon.com/certification/certified-cloud-practitioner/) exam (*CLF-C02*) is designed for individuals who want to demonstrate a broad understanding of the AWS Cloud. It is suitable for anyone looking to validate their overall knowledge of AWS services, regardless of technical expertise or specific job role.

The exam covers four main domains:

<div align="center">

| Domain                                   | % of Exam |
|:-----------------------------------------|:---------:|
| Cloud Concepts                           | 24%       |
| Security and Compliance                  | 30%       |
| Cloud Technology and Services            | 34%       |
| Billing, Pricing and Support             | 12%       |
| **Total**                                | **100%**  |

</div>

For the most up-to-date and official information about the exam, please refer to the [AWS Certification page](https://aws.amazon.com/certification/certified-cloud-practitioner/).
